# Проект 3 - Проект по БД и HH
Программа будет получать информацию о вакансиях с платформы hh.ru в России, спроектировать таблицы в БД PostgreSQL
и загрузить полученные данные в созданные таблицы.


## Разработка
В одноименных модулях расположены функции/методы/классы для реализации проекта

### Описание
Проект использует подклюение к БД(PostgreSQL). Убедитесь, что у Вас есть доступ и понимание работы с данным прилоежние
Шаблон данных для подключения к БД указан в файле database_example.ini
###
Проект создан студентом онлайн-университета Skypro https://my.sky.pro/


### Установка зависимостей
Для установки зависимостей, выполните команду:
```sh
poetry update
```
или
```sh
poetry install
```
